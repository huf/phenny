Installation &c.

1) Run ./phenny - this creates a default config file
2) Edit ~/.phenny/default.py
3) Run ./phenny - this now runs phenny with your settings
4) For the 'talk' module to work, you have to put cleverbot.py in the project folder, from https://code.google.com/p/pycleverbot/

Enjoy!
